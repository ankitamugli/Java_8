# Java 8 Programs

This repository contains Java 8 programs and examples that demonstrate the features introduced in Java 8.

## Topics Covered

- Lambda Expressions
- Functional Interfaces
- Stream API
- Method References
- Optional Class
- Date and Time API
- Collections Framework
- Predicate, Consumer, Supplier, and Function Interfaces
- Stream Operations (filter, map, reduce, collect)

## Project Structure

```
Java_8/
│
├── src/      # Java source files
├── bin/      # Compiled class files
├── lib/      # External libraries (if any)
└── README.md
```

## Technologies Used

- Java 8
- Eclipse IDE
- Git & GitHub

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/ankitamugli/Java_8.git
```

2. Open the project in Eclipse IDE.

3. Navigate to the `src` folder.

4. Run the desired Java program.
